# Arina Chernova's FYI Website


Arina Chernova's website is built using [Nikola Tesla Portfolio](https://github.com/iann-mathaiya/nikola-tesla.git) theme on top of Astro and uses [React Paper Shaders](https://github.com/paper-design/shaders). 

## Getting Started
Clone Repository
```sh
git clone https://github.com/iann-mathaiya/nikola-tesla.git
```

Install Dependencies
```sh
pnpm install
```

Development
```sh
pnpm run dev
```

Build
```sh
pnpm run build
```

Preview
```sh
pnpm run preview
```

## Deployment
The site is deployed on GitHub pages.
